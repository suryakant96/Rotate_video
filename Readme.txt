For this you should have python installed on your system.
Python Version 3.7

Library Required

opencv-python == 4.5.1.48

Install opencv-python
pip install opencv-python==4.5.1.48


How to execute file.
1. Save rotate.py in a folder
2. Open cmd prompt.
3. Change your directory to location to the location of rotate.py
4. Type python rotate.py
5. It will ask for the path for video you want to process.
6. Result file will store in the same directory in which your rotate.py was stored.
